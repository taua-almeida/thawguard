How scheduling works

A one-time window freezes a branch later without anyone on call to click the button.

  1. 1
    One-time windows

    A window covers a single stretch of time; it does not repeat.

  2. 2
    Exact branch names

    A scheduled window targets one repository branch.

  3. 3
    Automatic activation

    Thawguard activates due windows and recomputes {{ .RequiredContext }} statuses — the same path a manual freeze uses. A planned unfreeze lifts it automatically.
